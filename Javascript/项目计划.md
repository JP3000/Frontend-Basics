第 1 天：基础 + 类型（最容易丢分）
 
- let/const/var 区别
- 数据类型（7 种基本类型 + 引用类型）
- 类型判断：typeof、instanceof
- 浅拷贝 / 深拷贝
- 类型转换（== 与 === 坑点）
 
第 2 天：作用域、闭包、this（JS 灵魂）
 
- 全局/函数/块级作用域
- 作用域链
- 闭包（原理 + 用途 + 面试题）
- this 四种绑定规则
- call / apply / bind 手动绑定 this
 
第 3 天：原型、原型链、继承（最难但必考）
 
- prototype、proto
- 原型链查找规则
- 继承实现
- class 本质（语法糖）
- instanceof 原理
 
第 4 天：异步（重中之重）
 
- 回调函数 → 回调地狱
- Promise（全部 API）
- async/await
- 事件循环：宏任务 / 微任务
- 异步串行、并行
 
第 5 天：数组、对象、常用 API（写业务最快）
 
- 数组 10 个高频 API（map、filter、reduce…）
- 对象解构、展开运算符 ...
- 去重、扁平化、排序
- 字符串常用方法
 
第 6 天：DOM + 事件（前端基本功）
 
- 查询元素、增删改查
- 事件绑定、事件冒泡、捕获
- 事件委托
- 操作 class、style、属性
 
第 7 天：高频手写题（直接拉满竞争力）

- 防抖 / 节流
