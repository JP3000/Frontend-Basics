# JavaScript Experiments in This Folder

## 1) Install dependencies

This demo does not require extra dependencies.

## 2) Run basic entry file

npm start

## 3) Run algorithm experiment

npm run exp

## 4) Run in watch mode for fast iteration

npm run exp:watch

## 5) Run tests

npm test

## Suggested workflow

1. Edit experiments/playground.js
2. Save the file
3. Re-run npm run exp (or keep npm run exp:watch running)
4. Compare outputs and timings
