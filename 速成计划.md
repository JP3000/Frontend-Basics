# 三个月速成前端 + 找到大厂日常实习经验分享

## 第 1 个月：基础入门

### 1. HTML + CSS（7 天）
- 学习方式：参考【菜鸟教程】，边学边动手敲示例。
- 实践：花 3 天时间模拟京东 Web 端首页的静态页面。
- 建议：集中时间学习效率更高。

### 2. JavaScript + ES6（10 天）
- 学习资料：
  - 【菜鸟教程】文档
  - CSDN 上【轻松的小希】的博客
  - ES6 部分看 B 站【尚硅谷】的视频
- 难点：闭包、call/apply/bind、继承、Promise 等概念反复看才理解。
- 建议：不必一开始就深究，背八股时再彻底搞懂即可。

### 3. React 基础（14 天）
- 学习资料：B 站【李立超老师】的视频
- 实践：麦当劳案例耗时较多，建议时间紧张可跳过。
- 重点：掌握 `useState`、`useEffect`、`useRef`、`useCallback`、`useMemo`、Context 等基础 Hooks。
- 注意：Redux、RTK、RTKQ 部分初学时会一头雾水，实际做项目时才真正理解。

## 第 2 个月：项目实战

- 项目来源：B 站搬运的 YouTube 课程《Udemy - The Ultimate React Course 2024》
- 项目内容：酒店后台管理系统 + 预定官网
- 学习方法：
  - 不逐行跟敲，先由 AI 搭建框架
  - 自行思考功能实现方式
  - 遇到问题参考 GitHub 代码或询问 AI
  - 自行扩展项目功能，增强简历亮点
- 关键点：思考与解决问题的过程比简单跟敲更重要，能在面试中展示挑战与解决方案。
- 原理学习：
  - React 和 Next.js 原理部分仍参考李立超老师
  - Next.js 缓存与渲染机制参考掘金文章 + AI 辅助理解

## 第 3 个月：面试准备

### 1. 基础八股文
- 学习资料：【前端小圆】 + [vue3is.cn/interview](https://vue3is.cn/interview/)
- 方法：系统整理后，结合牛客网面经补充遗漏内容。

### 2. 项目八股文
- 深入理解项目中使用的技术，如认证系统：
  - 认证方式、优缺点、实现原理
  - 浏览器存储机制、XSS 与 CSRF 安全问题及解决方案
- 目标：延伸问题也要准备到位，体现深度。

### 3. JS 手写题、场景题、代码输出题
- 学习资料：【前端小圆】、掘金
- 重点关注：
  - 变量提升
  - 闭包
  - this 指向
  - 宏任务与微任务输出顺序
- 技巧：对不熟悉的写法可让 AI 重新整理代码。

## 求职 Tips

### 1. 提升到岗吸引力
- 在简历或打招呼时注明：
  - 【一周内可到岗】
  - 【一周可实习 5 天】
  - 【可实习 > 6 个月】

### 2. 简历内容精准
- 只写自己熟悉的内容
- 避免写“学过某课程”，除非能应对相关问题
- 建议写法：如“熟悉 http 等常见协议、tcp 三次握手流程”，而非“熟悉计算机网络”

### 3. 大厂日常实习看重：
- 基础扎实
- 学习能力强
- 沟通表达清晰
- 逻辑思维好
- 回答问题时建议结构：概念介绍 → 常用方法 → 使用场景
- 遇到不熟悉的问题可引导到自己熟悉的内容，或虚心表示会后续学习

### 4. 知识关联性
- 提到某个概念时，需准备其相关内容（如提到跨域方案中的 WebSocket，就要了解其原理）

### 5. 面试重点准备
- 自我介绍
- 项目中的挑战与解决方案
- 重点不在于代码细节，而在于：
  - 如何设计解决问题
  - 为何选择该方案
  - 体现思考过程
